## Inital setup ##

SETUP STEPS
1. install atom : Atom URL
- accept all presets
2. Clone repo
- press ctrl+shift+p
- search "clone"
- select github: clone
- paste in https link
- set location as "C:\go\src\github.com\michaeldcanady\Project01"
2. install go: GO URL
- accept all presets
3. install git bash: GIT URL
- set editor to Atom
- use Winodws' console window
4. use "go get" all packages in import packages list

FOR TEMPORARY USE

go run -tags="Liberty" github.com\michaeldcanady\Project01
